List of Chrome Search Tips

  • Use "site:" to search within a specific website
  • Use "define:" to search definitions
  • Use double quotes (" ") to search for an exact phrase
  • Use "filetype:" to search for results in a particular file type
  • Use AND, OR, NOT (represent by a minus sign)
  • Use "related:" to find similar websites
  • Use the wildcard asterisk *
  • Use "location:" to filter search results according to location
  • Use "before:" and "after:" to search within a date range
  • Use tilde operator to search for similar terms
  • Search using your voice with "Ok Goggle"
  • Use "cache:url" to find the cached version of the website
  • Use "intitle:" to search only in titles
  • Use google as your calculator
  • Use google advanced search
